# Збірка APK у хмарі (без Android Studio) — покрокова інструкція

Це працює навіть на 32-бітній Windows, бо сама збірка відбувається на
серверах GitHub, а не на твоєму комп'ютері. Тобі потрібен лише браузер і
безкоштовний акаунт.

## Крок 1: Реєстрація на GitHub

1. Відкрий https://github.com/join
2. Введи email, придумай пароль і ім'я користувача (username)
3. Підтверди email (прийде лист)

## Крок 2: Встанови GitHub Desktop (простий графічний застосунок)

1. Завантаж: https://desktop.github.com/
2. Встанови, відкрий, натисни **Sign in to GitHub.com** — увійди тим
   акаунтом, що щойно створив

## Крок 3: Створи репозиторій з твоєї папки проєкту

1. Розпакуй архів `my-messenger-mobile.zip` (якщо ще не розпакував)
2. У GitHub Desktop: **File → Add Local Repository**
3. Вкажи шлях до розпакованої папки `my-messenger-mobile`
4. Якщо застосунок скаже "This directory does not appear to be a Git
   repository" — натисни **create a repository** (посилання прямо в
   повідомленні)
5. Натисни **Create Repository**

## Крок 4: Опублікуй репозиторій на GitHub

1. У GitHub Desktop зверху буде кнопка **Publish repository**
2. Можеш залишити "Keep this code private" увімкненим (не обов'язково
   робити публічним)
3. Натисни **Publish Repository**

Це завантажить усі файли проєкту на GitHub, включно з
`.github/workflows/build-apk.yml` — файлом, який автоматично запускає
збірку APK в хмарі одразу після завантаження.

## Крок 5: Дивись, як йде збірка

1. Відкрий свій репозиторій на сайті github.com (GitHub Desktop має кнопку
   **View on GitHub**, або зайди вручну: github.com/твій-юзернейм/my-messenger-mobile)
2. Перейди на вкладку **Actions** (зверху)
3. Побачиш запущений процес "Build Android APK" з жовтим колом (виконується)
4. Зачекай 5-10 хвилин, поки колесо не стане зеленою галочкою ✅

Якщо буде червоний хрестик ❌ — щось пішло не так, скинь мені скріншот
логу (клацни на запис, розгорни кроки, де помилка).

## Крок 6: Завантаж готовий APK

1. Клацни на завершений запуск (той, що із зеленою галочкою)
2. Прокрути вниз до розділу **Artifacts**
3. Натисни на **my-messenger-apk** — завантажиться .zip
4. Розпакуй його — всередині файл `app-debug.apk`

## Крок 7: Встанови на телефон

1. Перекинь `app-debug.apk` на телефон (через USB, Google Диск,
   Telegram "Збережені повідомлення" собі, будь-як)
2. Відкрий файл на телефоні через файловий менеджер
3. Android попросить дозволити встановлення з невідомих джерел для
   застосунку, яким відкриваєш файл — дозволь
4. Встанови

## Якщо потім захочеш щось змінити в коді

1. Онови файли локально (я допоможу з кодом)
2. У GitHub Desktop з'явиться список змінених файлів
3. Внизу зліва напиши короткий опис зміни, натисни **Commit to main**
4. Натисни **Push origin** зверху
5. Це знову автоматично запустить збірку нового APK на вкладці Actions

## Пам'ятай про адресу сервера

У файлі `www/config.js` має бути правильна адреса твого `server.js`
(там же, всередині файлу, є коментарі, як дізнатись свою локальну IP через
`ipconfig`). Онови цей файл і зроби Commit + Push, якщо адреса сервера
зміниться.
